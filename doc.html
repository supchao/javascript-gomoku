示例代码

下面是一段示例AI代码，功能是随机走棋：


    var matrix = chessboard.getMatrix();
    var getRandomCoordinate = function () {
        var i = Math.floor(Math.random() * 15);
        var j = Math.floor(Math.random() * 15);
        return [i, j];
    };
    var coordinate = getRandomCoordinate();
    while (true) {
        if (matrix.getValueByCoordinate(coordinate) == 0) {
            break;
        } else {
            coordinate = getRandomCoordinate();
        }
    }
    chessboard.go(coordinate, this.color);

挑战者的AI代码执行环境，会得到一个chessboard变量，chessboard变量是Chessboard类型，它有两个最基础的方法getMatrix();和go(coordinate, color);

getMatrix();用来获得当前棋盘上棋子数据，它的返回值是Matrix类型，是对二维数组的封装

go(coordinate, color);用来往棋盘上摆放一个棋子，使用方法：go([0, 0], 'black');，这样调用意思是在棋盘的(0,0)点(左上角)，摆放一枚黑子，实际调用时，可以用环境初始化好的this.color替代'black'

重要说明：在系统中黑棋用数字3存储，白棋用数字1存储，空白用数字0存储。挑战者的代码在计算完毕一定要调用chessboard.go(coordinate, color);，如果不愿意使用Matrix类提供的方法，可以调用matrix.toArray();将矩阵转换成纯数据之后利用自己的方法取值和计算。

API文档

Class Chessboard

棋盘类，对棋盘的基础操作和获得棋盘状态

公有方法

go(Array coordinate, String color) : void 向棋盘上摆放棋子

getMatrix() : Matrix 获得棋盘当前状态数据

getStep() : Number 获取当前是第n步

Class Matrix

矩阵类，对二维数组的封装，用来存储棋盘状态

公有方法

getCoordinatesByValue(Number needle) : Array 获得此矩阵内所有值为needle的坐标，返回值是二维数组，例如[[0, 0], [2, 3]]

getValueByCoordinate(Array coordinate) : Number 获得给定坐标的值，例如getValueByCoordinate([1, 2]);

copy(interchange) : Matrix 复制一份当前的矩阵，如果将参数interchange设置为true，可将复制出的矩阵中1转成3，3转成1，0保持不变

toArray() : Number 将矩阵传换成数组
